# trans-rights-are-human-rights
A small website designed to point out the hypocrisy between how trans-rights and cis-rights are treated. This site will also have other LGBTQ+ rights noted as they often overlap.

Please feel free to open issues and submit pull requests for additions to this website. The intention is to have solid citations for every point on this site and, preferably to have multiple citations for every point.

Pages will have various prefix's depending on where the argument origin is from.

## Prefix List
Marriage-Page: This is for comparing various rights that are (often) only available through marriage.
GAC-Page: For generic topics of Gender Affirming Care that doesn't fall into other catagories.
GAC-Legal-Page: This is for comparison of Gender Affirming Care that is specifically dealing with legal rights.
GAC-Medical-Page: This is for comparison of Medical Gender Affirming Care such as HRT or Surgical Procedures.
